EXPLANATION
------------
This Angular project demonstrates a Template-Driven Form
built using Standalone Components (Angular 17+).

Fields:
- Name (text)
- Email (email)
- Course (dropdown)
- Gender (radio buttons)
- Submit button

When submitted, the entered details are displayed below the form.

main.ts EXPLANATION
-------------------
Instead of AppModule, Angular 17 uses bootstrapApplication() to
start the standalone AppComponent.

Code:
import { bootstrapApplication } from '@angular/platform-browser';
import { AppComponent } from './app/app';
bootstrapApplication(AppComponent);
